\section*{Формальная постановка задачи}


Имеется $N$ независимых работ, для каждой из которых задано время выполнения.
Требуется составить расписание выполнения этих работ без прерываний на $M$ идентичных процессорах,


Критерий К1: разбалансированность расписания


\textbf{Дано:}

\begin{itemize}
  \item $N$ — количество работ;
  \item $J = \{ j_1, j_2, \dots, j_N \}$ — множество работ;
  \item $\tau = \{ t_1, t_2, \dots, t_N \}$ — множество времён выполнения работ, где $t_i > 0$ для всех $i = 1, \dots, N$;
  \item $M$ — количество процессоров;
  \item $P = \{ p_1, p_2, \dots, p_M \}$ — множество процессоров.
\end{itemize}

\textbf{Определение расписания:}

Расписание задаётся булевой матрицей $HP \in \{0,1\}^{N \times M}$, где элемент $h_{ij}$ равен 1, 
если работа $j_i$ выполняется на процессоре $p_j$, и 0 в противном случае.

\textbf{Условия выполнения:}

Расписание должно удовлетворять следующим ограничениям:
\[
\sum_{i=1}^{N} \sum_{j=1}^{M} h_{ij} = N, \quad
\forall i \in [1, N] \; \sum_{j=1}^{M} h_{ij} = 1
\]
Это означает, что каждая работа выполняется ровно на одном процессоре.

\textbf{Критерий оптимальности:}

Пусть $G_j$ — упорядоченное множество индексов работ, назначенных на процессор $p_j$. 
Для каждого процессора $p_j$ общее время выполнения равно:
\[
T_j = \sum_{i \in G_j} t_i
\]
Тогда критерий $K_1$ (разбалансированность расписания) определяется как:
\[
K_1 = \max_{j \in [1,M]} T_j - \min_{j \in [1,M]} T_j
\]
Требуется найти расписание $HP$, минимизирующее данную величину:
\[
\min_{HP} K_1 = \min_{HP} \left( \max_{j \in [1,M]} T_j - \min_{j \in [1,M]} T_j \right)
\]

\textbf{Ограничения модели:}

\begin{itemize}
  \item На каждом процессоре в любой момент времени выполняется не более одной работы.
  \item Работы выполняются без прерываний.
  \item Переключение между работами на процессоре происходит мгновенно.
  \item Времена выполнения работ $t_i$ фиксированы и известны до начала составления расписания.
\end{itemize}
